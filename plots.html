Principal Component Analysis

PCA is a technique used to analyze the relationships between observations using dimension reduction. Dimension reduction is creating new variables as functions of existing variables. These new variables are called directions. This essentially reduces the number of variables in the dataset. PCAâ€™s are used because they provide detailed genetics data on many variables simultaneously. It also distinguishes clusters and populations which helps compare how related each observation is to each other.

Manhatten Plots

The Manhattan plot is effective at displaying large quantities of data points. Considering that there are approximately three billion nucleotides in a chromosome, Manhattan plots are an excellent choice to use for GWAS. When Manhattan plots are used for GWAS, normally Haplotypes represented by chromosomes are on the x axis. On the y-axis is the -log of the p-value. We use -log instead of log because -log makes the plot more comprehensible.
